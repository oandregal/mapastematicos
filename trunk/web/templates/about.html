Sobre el proyecto

MATE - mapastemáticos.org es una web para la visualización de información estadística. Es el resultado de 48h de programación para participar en abredatos.es, pasárnoslo bien y ver qué se puede hacer con poco tiempo y recursos con la información pública.

La idea principal del proyecto es mostrar la información pública disponible en el INE, compuesta de múltiples datos en bruto en formatos de tablas (csv, xls, …) de presentación poco intuitiva, a través de mapas y gráficos sencillos para aquellos datos susceptibles de ser localizados geográficamente posibilitando una mejor comprensión y accesibilidad de los mismos para los usuarios. Nuestro objetivo era crear una web donde se pudiesen visualizar mapas a partir de datos en crudo. Muchas cosas se nos han quedado en el tintero para cumplir el deadline del concurso, pero las iremos implementando una vez el jurado emita el veredicto del concurso
